
## Introducción

Como parte del proceso de análisis del mercado laboral en el área de desarrollo de software, se realizó una investigación de distintas ofertas de trabajo publicadas en portales de empleo como LinkedIn y otros sitios internacionales.

El objetivo de esta investigación es identificar los requisitos más comunes solicitados por las empresas, analizar las tecnologías más demandadas y definir un perfil profesional objetivo basado en la información recopilada.

Se analizaron **cinco ofertas laborales reales** relacionadas con el desarrollo web y desarrollo de software.

# Análisis de Patrones Comunes

Después de analizar las cinco ofertas laborales, se identificaron varios patrones en los requisitos del mercado laboral para desarrolladores.

Entre los más importantes se encuentran:

- Uso frecuente de **tecnologías web como HTML, CSS y JavaScript**.
    
- Uso de **frameworks modernos** como React, Angular o herramientas especializadas como WordPress y Webflow.
    
- Conocimiento de **control de versiones con Git**.
    
- Capacidad para trabajar con **APIs y servicios backend**.
    
- Conocimiento de **estructuras de datos y algoritmos**.
    
- Importancia de **trabajo en equipo y comunicación**.
    
- Uso de **metodologías ágiles**.
    
- Dominio del **idioma inglés** en entornos internacionales.
    

Estos patrones muestran que el mercado laboral busca desarrolladores que combinen habilidades técnicas sólidas con la capacidad de colaborar en entornos modernos de desarrollo.

# Perfil Profesional Objetivo

A partir del análisis realizado, el perfil profesional objetivo se orienta hacia el rol de **Desarrollador Web o Software Developer**.

Este perfil se enfoca en el desarrollo de aplicaciones web modernas utilizando tecnologías como:

- JavaScript y frameworks modernos (React o Angular)
    
- Desarrollo backend con Node.js o Python
    
- Integración de APIs y servicios web
    
- Uso de herramientas de control de versiones como Git
    

Además, se busca desarrollar habilidades de resolución de problemas, trabajo en equipo y comunicación efectiva.

# Justificación del Enfoque del Sitio Web

El sitio web desarrollado en este proyecto se orienta a la **presentación profesional para una postulación laboral ficticia en el área de desarrollo web**.

Esto se justifica debido a que muchas de las ofertas analizadas requieren que los candidatos presenten:

- Proyectos previos
    
- Portafolios de desarrollo
    
- Experiencia práctica en tecnologías web
    
- Capacidad de documentar y presentar su trabajo
    

Por esta razón, el sitio web funciona como un **portafolio profesional digital**, permitiendo demostrar habilidades técnicas y experiencia en desarrollo.

# Diferenciadores Clave

Durante el análisis del mercado laboral se identificaron dos factores que permiten destacar como desarrollador.

### Aprendizaje continuo

El sector tecnológico evoluciona rápidamente, por lo que la capacidad de aprender nuevas tecnologías y frameworks es fundamental para mantenerse competitivo.

### Combinación de habilidades técnicas y blandas

Las empresas valoran desarrolladores que no solo sepan programar, sino que también puedan comunicarse efectivamente, trabajar en equipo y resolver problemas de manera eficiente.
